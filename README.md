# Leaflet Rotas

Teste com a biblioteca Leaflet e Leaflet Routing Machine para criar rotas no mapa.

## Funcionalidades

- Rota entre dois pontos
- Pinos arrastáveis
- Rotas alternativas
- Linha tracejada
- Botão para apagar a rota

## Uso

1. Instale as dependências:
```bash
npm install leaflet
npm install leaflet-routing-machine
